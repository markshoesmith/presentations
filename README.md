## Repository Structure

Each presentation lives in its own folder named `YYYY-MM_conference_topic` and contains:

```
YYYY-MM_conference_topic/
├── README.md        # Abstract, bio, links to recording/event page
├── slides.pptx      # Main slide deck (PowerPoint)
├── slides.pdf       # Exported PDF version of the slides
└── resources/       # Code samples, demos, supplementary material
```
